# 🤖 AI & Machine Learning Mini Projects

Welcome to my **AI & Machine Learning Mini Projects** repository!

This repository contains beginner-friendly Machine Learning projects developed using **Python**, **Scikit-learn**, **Pandas**, and **Matplotlib**. These projects demonstrate the complete Machine Learning workflow, including data preprocessing, visualization, model training, evaluation, and prediction.

---

## 📚 Projects Included

### 📈 1. Student Performance Predictor

Predicts students' exam scores based on study hours using **Linear Regression**.

**Key Features**
- Data Loading & Preprocessing
- Data Visualization
- Linear Regression Model
- Model Evaluation
- Score Prediction

---

### 🌸 2. Iris Flower Classifier

Classifies Iris flowers into different species using the **K-Nearest Neighbors (KNN)** algorithm.

**Key Features**
- Data Preprocessing
- Feature Scaling
- KNN Classification
- Confusion Matrix
- Classification Report
- Accuracy Evaluation
